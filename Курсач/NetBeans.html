NetBeans

NetBeans Это IDE, которое представляет собой бесплатную интегрированную среду разработки с открытым кодом для разработчиков программного обеспечения. Среда IDE NetBeans предоставляет все средства, необходимые для создания профессиональных приложений рабочей среды, корпоративных, мобильных и веб-приложений на языках Java, C/C++, а также на других динамических языках. Среда IDE может работать на различных платформах, включая операционные системы Windows, Linux, Solaris и Mac, отличается простотой установки и удобством использования и не требует дополнительной настройки. Проект NetBeans IDE поддерживается и спонсируется компанией Oracle, однако разработка NetBeans ведётся независимым сообществом разработчиков-энтузиастов (NetBeans Community) и компанией NetBeans Org. В сентябре 2016 года Oracle передала интегрированную среду разработки NetBeans в руки фонда Apache. NetBeans полностью открыта для вклада разработчиков, желающих усовершенствовать платформу.

Из плюсов, платформа содержит:
  • Умное редактирование и автодополнение кода;
  • Настраиваемые сочетания клавиш для более быстрой работы;
  • Визуальный отладчик;
  • Интеграция с Git, Maven и другими платформами.
  • Малый обьём;
  • Простота в использовании
    • Из минусов можно выделить:
  • Отсутствие встроенной интерактивной системы справки по тому языку, для которого это IDE предназначено
  • В некоторых случаях авто-завершение текста предлагает совсем не то что нужно
  • Несколько большое потребление памяти ввиду использования Java

    • Поддерживаемые платформы: Windows/macOS/LinuxOS/Solaris
